# 🐲 ConvertDragon (변환해용) - High-Performance Image Converter

**[무료로 이미지 변환하기 (https://convertdragon.com)](https://convertdragon.com)**

ConvertDragon은 사용자 편의와 프라이버시를 최우선으로 생각하는 웹 기반 이미지 프로세싱 서비스입니다. 복잡한 설치 없이 브라우저에서 즉시 이미지를 변환하고 압축하세요.

## 🚀 Key Business Features
- **100% Client-Side Processing**: 모든 이미지 처리는 사용자의 브라우저 내에서 이루어집니다. 이미지가 서버로 업로드되지 않아 보안이 완벽하며, 매우 빠릅니다.
- **HEIC to JPG/PNG**: 아이폰 최신 포맷인 HEIC를 윈도우나 웹에서 바로 사용할 수 있도록 고품질로 변환합니다.
- **Web Optimization**: 이미지 용량을 획기적으로 줄여 웹사이트 로딩 속도를 개선하고 SEO 점수를 높여줍니다.
- **No Install, No Sign-up**: 회원가입이나 설치 없이 모든 기능을 무료로 제공합니다.

## 🛡️ Privacy & Security
본 서비스는 사용자의 데이터를 수집하거나 저장하지 않습니다. 
- 서버 저장 zero: 이미지 프로세싱 완료 후 메모리에서 즉시 제거됩니다.
- 개인정보 보호: 별도의 트래킹 없이 익명으로 안전하게 서비스를 이용할 수 있습니다.

## 🛠 Tech Stack (Commercial Grade)
- **Frontend**: Nuxt 3 (Vue.js) for high-performance SSR/SSG.
- **Styling**: Tailwind CSS for responsive and modern UI.
- **Deployment**: Vercel for 99.9% uptime and global edge delivery.
